# less-ayx-journal

## 简介

`less-ayx-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅用于存放和展示一些静态 HTML 内容，方便长期保存与共享。

## 目录说明

```
.
├── index.html          # 仓库入口页面
├── pages/              # 存放主要 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源文件
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：按编号或日期命名的独立 HTML 页面。
- `assets/`：页面共用的样式、脚本及图片资源。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开。
- 页面内容可能包含文本、图片、链接等常见网页元素，均为静态展示用途。
- 部分页面可能引用 `assets/` 目录下的资源，打开时请确保相对路径正确。

## 维护说明

- 本仓库不定期更新，新增页面会按顺序放入 `pages/` 目录。
- 不保证所有页面长期可访问或兼容所有浏览器版本。
- 欢迎通过 Issues 提交问题或建议，但请注意本仓库不针对任何具体网站。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 如需部署到静态托管服务，可直接将整个仓库作为站点根目录。

## 许可

本仓库内容仅供个人学习与归档使用，不提供任何明示或暗示的保证。
